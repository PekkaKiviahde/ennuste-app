# UI – One-screen v1

## Periaate
- Yksi ruutu, minimiklikit.
- Kaksi tilaa työpaketille:
  1) SETUP (baseline ei lukittu)
  2) TRACK (baseline lukittu)
- UI-tekstit terminology_get_dictionary(org, locale).

## SETUP (baseline-asetus)
- Perustiedot: nimi, kuvaus, lead-littera
- Koostumus: lisää/poista litteroita + “lisää item-koodista” (vain tavoitearviorivit, hyväksyntäketju)
- Taloudellinen suunnitelma: toteutustapa + hankinta/riskihuomiot (minimi)
- Baseline-esikatselu + lukituspyyntö (PM 1/2, tuotantojohtaja 2/2)

## TRACK (seuranta)
- Viikkopäivitys: % + haamut + memo (esitäytöt)
- Kuukausiennuste: muutos + perustelu + hyväksynnät
- Selvitettävät: inbox (unmapped actuals) -> liitä työpakettiin / luo uusi
